movie 테이블에서 제목을 검색하는 PL/SQL 블록을 작성하시오.

조건

입력 키워드 v_key가 주어진다.

대소문자를 구별하지 않고 검색한다.

v_like = 'Y' 이면 LIKE 검색을 한다.

“제목에 키워드가 포함된” 영화를 찾는다. (즉, %키워드%)

단, 사용자가 v_key에 % 또는 _를 입력하더라도 와일드카드로 해석하지 말고 문자 그대로 검색되도록 처리한다.

예: v_key = '%' 이면 제목에 % 문자가 들어있는 영화만 검색되어야 함

예: v_key = '_' 이면 제목에 _ 문자가 들어있는 영화만 검색되어야 함

결과는 title (year) 형태로 모두 출력하고, 없으면 "검색 결과 없음" 출력
